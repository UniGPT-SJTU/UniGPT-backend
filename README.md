## 运行
Linux 和 MacOS:
```
./mvnw spring-boot:run
``` 
Windows:
```
./mvnw.cmd spring-boot:run
```
后端URL为http://localhost:8080
## 构建
Linux 和 MacOS:
```
./mvnw package
```
Windows:
```
./mvnw.cmd package
```
